# Snake Game

This repository contains a simple terminal-based Snake game implemented in Python using the `curses` library.

## How to Run

1. Ensure you have Python installed (version 3.x).
2. The game relies on the built-in `curses` module, which is typically available on Unix-like systems.
3. Run the game from your terminal:

```bash
python snake.py
```

Use the arrow keys to control the snake. The game ends when the snake hits the wall or itself.
